#### **Shop-Site** це інтернет-магазин, в якому можна:
- **Зареєструватися**
- **Війти** до свого акаунту
- **Придбати** товари
- **Видалити** або **додати** товар (якщо ви **адміністатор**)
- **Зв'язатися** з продавцем
